#### 流程

1. 从数列中选出一个枢纽元。
2. 将枢纽元置于最右边。
3. 将小于枢纽元的放在左边。
4. 将枢纽元置于正确位置。
5. 对左右区间重复第二步，直到每个区间少于两个元素。

---

#### 实现

```cpp
void qSort(std::vector<int> &nums, int left, int right)
{
    static std::default_random_engine random_engine;

    if (left >= right)
        return;

    // 随机一个元素作为枢纽元，将其置于最右边
    size_t pivot_index = random_engine() % (right - left + 1) + left;
    std::swap(nums[pivot_index], nums[right]);

    // 将小于枢纽元的数都放在左边
    for (int i = pivot_index = left; i < right; ++i)
    {
        if (nums[i] < nums[right])
        {
            std::swap(nums[pivot_index], nums[i]);
            pivot_index += 1;
        }
    }

    // 最后将枢纽元置于正确位置
    std::swap(nums[pivot_index], nums[right]);

    // 枢纽元已经在正确位置，无需再进行排序
    qSort(nums, left, pivot_index - 1);
    qSort(nums, pivot_index + 1, right);
}

void quickSort(std::vector<int> &nums)
{
    qSort(nums, 0, nums.size() - 1);
}
```
