###### 隐式参数

定义成员函数时，会隐式将`this`作为第一个形参。调用成员函数时，会隐式将对象地址作为`this`传递。

```cpp
struct test
{
    auto call() -> void { std::cout << "call\n"; }
};

auto main() -> int
{
    test t;
    std::invoke(&test::call, &t);   // call
    
    return 0;
}
```

###### const

const属于函数原型的一部分，函数定义中必须包含。

const对象只能调用const成员函数。non-const对象优先调用non-const成员函数。
