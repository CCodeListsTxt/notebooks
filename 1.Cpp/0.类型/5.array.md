###### 未知范围数组

如果需要接受未知范围的数组作为参数，有两种方式实现：

* 使用模板+引用传递。

  ```cpp
  template <int M> 
  auto call(int (&arr)[M]) -> void
  {
      for (int num : arr)
          std::cout << num << " ";
  }
  ```
  
* 使用std::span。

  ```cpp
  auto call(std::span<int> arr) -> void
  {
      for (int num : arr)
          std::cout << num << " ";
  }
  ```
