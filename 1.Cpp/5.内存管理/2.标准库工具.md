###### std::to_address

to_address()提供简单且安全的方式获取原始指针，可传入原始指针、智能指针以及部分迭代器。

```cpp
auto main() -> int
{
    std::shared_ptr<int> smart_p;
    std::vector<int> vec{1, 2, 3};

    std::cout << std::to_address(smart_p) << "\n";      // nullptr
    std::cout << std::to_address(vec.begin()) << "\n";

    return 0;
}
```
