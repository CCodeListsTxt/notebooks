#### 别名模板

```cpp
template <typename T>
using Vec = std::vector<T, MyAlloc<T>>;

int main()
{
    Vec<int> v;
    return 0;
}
```

#### 模板模板参数

```cpp
// Type是一个模板类型
template <
	template <typename, typename> typename Type>
void call() { printf("%s\n", typeid(Type).name()); }

int main()
{
	call<std::vector>();	// std::vector

	return 0;
}
```
