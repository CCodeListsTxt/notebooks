C++20引入了三向比较运算符`<=>`。其返回表示大小的状态，该状态可以和整数比较。

如果类定义了`<=>`运算符，编译器将自动合成六个基本比较运算符。

```cpp
struct Person
{
    std::string name;
    uint8_t age;

    auto operator<=>(const Person &rhs) const
    {
        return this->age <=> rhs.age;
    }
};

int main()
{
    std::cout << (Person{"", 20} > Person{"", 10}); // true

    return 0;
}
```
