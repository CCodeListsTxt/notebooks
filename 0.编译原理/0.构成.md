1. scanner，词法分析。

   输入：字节序列，如a[index] = 4 + 2

   输入：token序列，如

   ```txt
   a		identifier
   [		left bracket
   index	identifier
   ]		right bracket
   =		assignment
   4		number
   +		plus sign
   2		number
   ```

2. parser，语法分析。

   输入：token序列。

   输出：syntax tree。

3. semantic analyzer，语义分析。

   输入：抽象语法树。

   输出：带类型信息和符号表的syntax tree。
