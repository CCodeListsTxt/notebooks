#### 基本数据类型

| 数据类型        | 说明                                                         |
| --------------- | ------------------------------------------------------------ |
| `smallint`      | 小整形。                                                     |
| `int`           | 整形。                                                       |
| `numeric(p, d)` | 定点数，加上符号位有`p`个数字，其中`d`个数字表示小数。<br />如`numeric(3, 1)`可以存储44.5，但不能存储-44.5或4.45。 |
| `real`          | 单精度浮点。                                                 |
| `double`        | 双精度浮点。                                                 |
| `float(n)`      | 精度至少为`n`位的浮点。                                      |
| `char(n)`       | 固定长度字符串。如果将`"abc"`存入`char(10)`类型，那么会在`"abc"`后面加上7个空格以满足10个字符。 |
| `varchar(n)`    | 可变长度字符串。                                             |
