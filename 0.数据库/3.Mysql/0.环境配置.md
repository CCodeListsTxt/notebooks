#### WSL配置mysql

1. 安装`mysql-server`。

   ```shell
   > sudo apt install mysql-server
   ```

2. 启动`mysql`服务。

   ```shell
   > sudo service mysql start 
   ```

3. 添加访问权限。

   ```shell
   > sudo chmod 777 -R /var/run/mysqld /var/lib/mysql /var/log/mysql
   ```

4. 使用`root`进入`mysql`并添加普通用户。

   ```shell
   > sudo mysql -uroot -p
   ```

   ```mysql
   create user 'usr'@'localhost' identified by 'pass';
   ```
