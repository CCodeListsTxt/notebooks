###### 编译选项

| 选项           | 说明                           | 可选值                                           |
| -------------- | ------------------------------ | ------------------------------------------------ |
| `-f <format>`  | 输出文件格式                   | `bin`、`elf`、`elf32`、`elf64`、`win32`、`win64` |
| `-o <out>`     | 指定输出文件名                 |                                                  |
| `-l <list>`    | 生成包含汇编和字节码的列表文件 |                                                  |
| `-I <include>` | 添加包含头文件的搜索路径       |                                                  |
| `-g`           | 生成调试信息                   |                                                  |
| `-O number`    | 优化等级                       | `-O0`、`-O1`、`-Ox`                              |
