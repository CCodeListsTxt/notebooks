使用`enum`关键字定义枚举类，其中各个枚举常量用逗号分割，且枚举常量必须定义在其他字段和方法之前。

所有的枚举类默认继承自`Enum`，可以使用`values()`返回所有枚举常量。

```java
public class App {
    public static void main(String[] args) {
        for (var color : Color.values()) {
            System.out.println(color);
        }
    }

    enum Color {
        RED, GREEN, BLUE;
    }
}
// RED
// GREEN
// BLUE
```
